## El Proyecto
Una herramienta experimental pero funcional diseñada para actuar como un "guardián" del código durante el ciclo de desarrollo, utilizando modelos de lenguaje avanzados para detectar patrones inseguros.

## Solución Técnica
El sistema analiza localmente la estructura del código y los archivos clave para identificar posibles riesgos de seguridad, dependencias desactualizadas o malas prácticas arquitectónicas.

## Capacidades de IA
- Escaneo de vulnerabilidades local asistido por LLM.
- Generación de reportes de seguridad en lenguaje natural.
- Sugerencias automáticas de refactorización para mitigar riesgos detectados.

## Impacto
Reduce la carga cognitiva de los desarrolladores al proporcionar una capa extra de revisión automática antes de las auditorías de seguridad formales.
