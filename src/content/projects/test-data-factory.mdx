## Problem
Las pruebas dependían de datos creados manualmente, lo que hacía difícil reproducir fallas entre ambientes y equipos.

## Solution
Servicio de generación de datasets versionados por tipo de flujo (happy path, edge cases, permisos, fallas controladas).

## What I did
- Diseñé un catálogo de escenarios reutilizables para API y E2E.
- Implementé endpoint de provisionamiento con seeds determinísticas.
- Definí limpieza y expiración de datos para evitar contaminación entre pruebas.
- Integré el servicio con pipelines de smoke y regression.

## Why it matters
Mejora repetibilidad de pruebas, acelera diagnóstico y reduce tiempo perdido en setup manual.

## Human vs AI contribution
- Human: diseño de modelo de datos y contratos de escenarios.
- AI: apoyo en esqueletos de endpoints y ejemplos de documentación.
