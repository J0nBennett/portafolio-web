## El Proyecto
Este portafolio es más que una carta de presentación; es un proyecto de ingeniería que demuestra mi capacidad para construir interfaces modernas, rápidas y accesibles.

## Rediseño Visual
Recientemente actualizado con un **Dark Theme Premium**, utilizando:
- **Glassmorphism:** Para darle profundidad a las interfaces.
- **Gradientes Dinámicos:** Enfatizando la marca personal.
- **Animaciones Entrance:** Para una experiencia de usuario fluida y profesional.

## Arquitectura
Basado en Astro v5 para lograr un rendimiento impecable e islas de interactividad donde son necesarias, manteniendo el resto del sitio estático y veloz.
