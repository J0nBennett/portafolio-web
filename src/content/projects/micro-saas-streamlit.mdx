## El Proyecto
Exploración de la viabilidad de productos de software enfocados en nichos específicos, utilizando el ecosistema de Python para construir interfaces funcionales en tiempo récord.

## Enfoque
Streamlit permite pasar de un script de análisis de datos a una aplicación web interactiva en cuestión de horas, lo que es ideal para validar MPVs con usuarios reales rápidamente.

## Tecnologías
- Python para la lógica de negocio y procesamiento de datos.
- Streamlit para el frontend reactivo.
- Integración con bases de datos ligeras para persistencia de estado.
