## El Proyecto
Una herramienta diseñada para entusiastas del running que necesitan centralizar sus planes de entrenamiento y seguir su progreso de manera visual.

## Características
- Dashboard de progreso semanal.
- Gestión de rutas y objetivos de distancia.
- Interfaz altamente reactiva construida con los últimos estándares de React.

## Aprendizaje
Este proyecto sirvió para profundizar en la arquitectura de estados complejos en React y el uso de Vite para una experiencia de desarrollo extremadamente rápida.
